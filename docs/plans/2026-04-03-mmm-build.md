# MMM Implementation Plan

> For Hermes: Use subagent-driven-development skill to implement this plan task-by-task.

Goal: Build Monday Music Mix as a GitHub Pages-ready static music archive with import, publish, and weekly draft-generation workflows.

Architecture: Use a data-first repository with JSON content under data/, Node-based static site generation for GitHub Pages, and Python automation scripts for import/publish/draft generation. Keep the site fully buildable from local data while making AI-assisted weekly generation optional via environment variables.

Tech Stack: Node.js, vanilla static generation scripts, Python 3, GitHub Actions, JSON Schema, GitHub Pages.

---

Task groups:
1. Data model + Tumblr import + archive seed
2. Static site scaffold + shared templates + route generation
3. Publish workflow + weekly draft generation + CI/deploy
4. Integration, local validation, and GitHub repo setup
